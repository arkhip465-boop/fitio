FITIO WEB / iPHONE

Готовая папка для бесплатной публикации как веб-приложение.

После публикации друг на iPhone:
1. Открывает ссылку в Safari.
2. Нажимает «Поделиться».
3. Нажимает «На экран Домой» / «Add to Home Screen».
4. Открывает fitio. с иконки.

Важно: для установки и офлайн-режима сайт должен открываться по HTTPS.
GitHub Pages подходит.
